# zfile 下载/同步工具

> https://github.com/zfile-dev/zfile  

* 将 zfile 的文件下载/同步到本地
* 支持遍历 zfile 磁盘
* 支持增量同步, 已遍历的文件夹会缓存到本地, 下次遍历时会自动跳过, 删除即可重新同步
